# react-ts

React Scaffolding

React、Redux、Router、Typescript、Webpack、Eslint、Tslint

## 构建

```
npm install       # 安装依赖模块
```

```
npm run start     # 启动开发环境，默认监听4000端口
```

```
npm run build     # 生产环境正式打包，会自动生成dist文件夹
```

```
npm run clean     # 清除dist文件夹
```

## 特性

* HMR热替换

* 基于react-redux-starts-kit

* 默认加入了Antd
